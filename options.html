Send to Me

Configure destinations for the current tab URL. You can add any number of targets — clicking the extension icon sends the URL to all of them at once. If nothing is configured, a QR code is shown instead.

Telegram

Create a bot with @BotFather, paste the token, and set the chat ID you want to send to. To find your chat ID, message @userinfobot. For group topics, also set the topic ID.

Webhooks

The extension POSTs JSON { url, title, timestamp } to each URL.

Settings are synced via your Chrome profile.
